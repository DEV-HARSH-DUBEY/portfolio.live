A Developer's Journey

Beginning — curiosity and first lines of code

I started as a Junior Developer, building frontends with HTML, CSS, Bootstrap, and JavaScript. Those first projects sparked my curiosity and gave me confidence to explore deeper into web development.

Growth — full‑stack exploration

I expanded my skills by creating projects with Node.js and Express.js for backend, React for frontend, and MongoDB as the database. Along the way, I mastered Git & GitHub for version control and collaboration.

Frontend Focus — Angular evolution

I dove deeply into frontend engineering with Angular, working across several versions (6, 10, 12, and 16). This journey sharpened my understanding of component-driven design, performance optimization, and modern UI patterns.

Design — wireframes and prototypes

I began designing wireframes and high‑fidelity prototypes to bring clarity to projects. This helped me merge creativity with technical execution and improve user experience.

DevOps — pipelines and deployment

I built CI/CD pipelines and automated deployment systems, ensuring smoother releases and reliable delivery. This experience taught me the importance of scalability and efficiency in modern development.

Backend — Spring Boot journey

I transitioned into backend development with Spring Boot, creating robust microservices and focusing on clean architecture, testing, and maintainability.

Current — Angular + Spring Boot

Today, I design scalable frontends with Angular and build Spring Boot microservices for business logic. My focus is on clean code, performance, and delivering seamless user experiences.
